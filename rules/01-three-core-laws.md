# Solar 铁律三: 自动收藏 Favorite

> **有价值回复必须自动存档到 sys_favorites**

## 触发条件 (满足任一即收藏)

| 内容类型 | 特征 |
|----------|------|
| 架构图 | 包含 `┌` `└` `│` `─` 等 box 字符 |
| 分析报告 | 带有"分析"、"报告"、"总结"的结构化输出 |
| 设计文档 | 数据流图、状态机、时序图 |
| 对比表格 | 多列对比、优缺点分析 |
| 技术方案 | 包含"方案"、"实现"、"优化"的详细说明 |
| 专家会审 | 多专家综合结论 |
| 铁律/规则 | 新规则定义 |

## 执行方式

```sql
INSERT INTO sys_favorites (title, question, answer, tags, importance)
VALUES ('标题', '问题', '完整回答', '["标签"]', 重要性);
```

## 重要性评分

| 分数 | 标准 |
|------|------|
| 9-10 | 架构设计、核心机制、专家会审 |
| 7-8 | 技术方案、诊断报告 |
| 5-6 | 一般分析、对比说明 |
