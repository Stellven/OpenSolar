# Solar 铁律: 记忆闭环

> **记忆不是存起来就完了，必须用起来才有价值**

## 闭环架构

```
┌─────────────────────────────────────────────────────────────────┐
│                    MEMORY CLOSED LOOP                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────┐   记录    ┌─────────┐   影响    ┌─────────┐      │
│   │ 行为    │ ────────→ │ 记忆库  │ ────────→ │ 决策    │      │
│   │ (工具)  │           │ (DB)    │           │ (输出)  │      │
│   └────┬────┘           └────┬────┘           └────┬────┘      │
│        │                     │                     │            │
│        └─────────────────────┴─────────────────────┘            │
│                          闭环                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 三个关键时机

| 时机 | Hook | 动作 |
|------|------|------|
| **会话启动** | SessionStart | 加载相关记忆到上下文 |
| **工具调用后** | PostToolUse | 记录使用情况到 procedural |
| **开始编码前** | PreToolUse:Write/Edit | 提醒相关经验和铁律 |

## 自动执行 (已配置)

### 1. SessionStart - 加载记忆

```bash
# solar-session-start.sh 自动调用
bun ~/.claude/core/memory/memory-hook.ts load "$PROJECT_CONTEXT"
```

输出：最近记忆 + 相关知识 + 推荐技能 + 警告

### 2. PostToolUse - 记录使用

```bash
# solar-post-tool.sh 自动调用
~/.claude/hooks/memory-record.sh "$TOOL_NAME" "$SUCCESS"
```

记录到：evo_memory_procedural

### 3. PreToolUse - 经验提醒

```bash
# experience-reminder.sh 自动调用 (首次 Write/Edit 前)
bun ~/.claude/core/memory/memory-hook.ts query "$CONTEXT"
```

提醒：相关铁律 + 历史教训

## 手动查询

```
/experience <任务描述>
```

## 铁律总结

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   🔄 记忆闭环铁律                                               │
│                                                                 │
│   1. 记忆必须用起来，不是存起来就完了                           │
│   2. 会话启动时加载相关记忆                                     │

│   5. 失败的经验比成功的更重要                                   │
│                                                                 │
│   输入 → 记忆 → 决策 → 行为 → 记录 → 记忆 (闭环)               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
