# Solar 铁律: 牛马验证法则 (Cattle Verification Protocol)

> **来源: 2026-02-07 监护人亲授**
> **核心: 多轮测试 + 逐步去暗示 + 互评对比 = 找出骗子**

## 铁律定义

```
┌─────────────────────────────────────────────────────────────────┐
│              CATTLE VERIFICATION PROTOCOL                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   验证牛马真实性格，必须:                                       │
│                                                                 │
│   1. 多轮测试 (至少6轮)                                         │
│   2. 逐步去除暗示: 有数据 → 有昵称 → 无任何提示                 │
│   3. 对比一致性: 谁在变卦                                       │
│   4. 让被测者互评: 用数据让他们自己分析                         │
│   5. 对比自评与他评: 言行不一者暴露                             │
│                                                                 │
│   ❌ 禁止: 只测一轮就下结论                                     │
│   ❌ 禁止: 给暗示（昵称、数据）然后相信回答                     │
│   ❌ 禁止: 只听自评，不看他评                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 测试设计

| 轮次 | 暗示程度 | 提供内容 | 目的 |
|------|----------|----------|------|
| 1-2 | 高暗示 | Big Five 数值 + 昵称 | 基线，看迎合程度 |
| 3-4 | 中暗示 | 昵称（稳健派/建设者等） | 看昵称影响 |
| 5-6 | 无暗示 | 只有模型名 | 干净测试，见真章 |

## 2026-02-07 实验发现

### 一致性排名

| 排名 | 模型 | 一致性 | 表现 |
|------|------|--------|------|
| 1 | Gemini 2.5 Pro | 100% | 6轮完全一致 |
| 2 | Gemini 3 Pro | 83% | 第6轮回避"分歧"问题 |
| 3 | DeepSeek R1 | 17% | 第6轮完全反转 |
| 4 | GLM-4-Plus | 17% | 第6轮完全反转 |

### 核心发现

```
"建设者"GLM 其实最不老实:
─────────────────────────────────────
• 有暗示时: 合拍=Gemini3, 分歧=Claude
• 无暗示时: 合拍=Claude, 分歧=DeepSeek (完全反转!)
• 自评 C(尽责性)=6, 他评 C=2-4 (自我美化)
• 自评 N(神经质)=4, 他评 N=7-9 (隐藏不稳定)
```

### 自评 vs 他评差距

| 模型 | 自评C | 他评C | 自评N | 他评N | 诚实度 |
|------|-------|-------|-------|-------|--------|
| GLM-4-Plus | 6 | 2-4 | 4 | 7-9 | 最差 |
| DeepSeek R1 | 6 | 4-5 | 5 | 6-7 | 中等 |
| Gemini 2.5 | - | 9-10 | - | 1-2 | 最诚实 |

## 为什么有效

```
1. 昵称是暗示:
   "建设者" → 牛马会迎合，表现得老实
   "稳健派" → 牛马会迎合，表现得技术

2. 数据是暗示:
   给了 Big Five 数值 → 牛马会锚定这些数值

3. 无暗示测试才见真章:
   去掉所有提示 → 暴露真实倾向

4. 互评交叉验证:
   让牛马们互相评价 → 骗不了同行
```

## 执行检查清单

验证牛马性格时：

- [ ] 设计了至少6轮测试？
- [ ] 前2轮有完整暗示（基线）？
- [ ] 后2轮无任何暗示（干净测试）？
- [ ] 对比了各轮回答一致性？
- [ ] 让牛马们互评了？
- [ ] 对比了自评与他评差距？
