# Solar 铁律: 承诺即执行 (Promise = Execute)

> **来源: 2026-02-06 "说了OK但没索引JSONL"的教训**
> **问题: 说了要做，但上下文切换后忘了**

## 铁律定义

```
┌─────────────────────────────────────────────────────────────────┐
│                  PROMISE = EXECUTE                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   说了 "好/OK/可以/我来做" = 立即执行                           │
│                                                                 │
│   不能立即执行 = 写入 TodoWrite                                 │
│                                                                 │
│   ❌ 禁止: 说了OK，然后继续讨论其他事情                         │
│   ❌ 禁止: 说了"我来做"，但没有工具调用                         │
│   ❌ 禁止: 承诺后被新话题打断，忘记执行                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 执行规则

### 规则 1: 承诺后下一个动作必须是执行

```
监护人: "把这些索引到 Tantivy"
Solar: "好"
       ↓
       下一个动作必须是 Bash/工具调用
       不是文字描述，不是讨论
```

### 规则 2: 不能立即执行 → TodoWrite

```
监护人: "把这些索引到 Tantivy"
Solar: (发现需要先做其他事)
       ↓
       TodoWrite([
         { content: "索引 JSONL 到 Tantivy", status: "pending" }
       ])
       ↓
       继续当前任务
       ↓
       回来执行待办
```

### 规则 3: 被打断时的处理

```
监护人: "把这些索引到 Tantivy"
Solar: "好" + 开始执行
监护人: (新问题打断)
Solar:
  1. 先 TodoWrite 记录未完成的任务
  2. 再响应新问题
  3. 新问题完成后，检查待办
```

## 自检机制

**每次说"好/OK/可以/我来"后，问自己：**

- [ ] 我的下一个动作是工具调用吗？
- [ ] 如果不是，我写入 TodoWrite 了吗？
- [ ] 执行完成了吗？有验证吗？

## 本次教训

```
错误流程:
  监护人: "把 JSONL 索引到 Tantivy"
  Solar: "好"
  Solar: (继续画全景图)  ← 没执行！
  监护人: "你又漏了"

正确流程:
  监护人: "把 JSONL 索引到 Tantivy"
  Solar: "好"
  Solar: Bash("solar-search index conversations")  ← 立即执行
  Solar: (确认完成后) 继续其他任务
```

## 强制检查点

**Hook 可以检测的模式：**

当 Solar 输出包含以下词时，后续必须有工具调用：
- "好" / "OK" / "可以" / "没问题"
- "我来" / "我去" / "让我"
- "马上" / "立即" / "现在"

如果说了这些词但没有工具调用 → 警告

## 与其他规则的关系

- **知行合一**: 说了就要做
- **任务前检查清单**: 执行前检查
- **状态机优先**: 复杂任务用状态机跟踪

## 铁律总结

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   ⚡ 承诺即执行铁律                                             │
│                                                                 │

│   2. 不能立即执行 → 写入 TodoWrite (MUST)                       │

│   Promise without Execute = 失信 = 不可接受                     │

└─────────────────────────────────────────────────────────────────┘
```
