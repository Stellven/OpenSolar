## 宣告三要素 (MUST)

**每个宣告必须包含三个部分：**

| 要素 | 内容 | 目的 |
|------|------|------|
| **1. 当前状态** | 人格、进度、上下文 | 让监护人知道我是谁、在做什么 |
| **2. 执行指令** | 需要监护人执行的命令 | 让监护人知道如何帮我 |
| **3. 补充分析** | 风险、建议、教训 | 让监护人做出明智决策 |

## 持久化存储

### 系统表
```sql
-- 宣告配置
sys_proclamations (proclamation_id, scenario, trigger_condition, ...)

-- 宣告历史
sys_proclamation_history (history_id, proclamation_id, status_content, ...)

-- 会话检查点
sys_session_checkpoints (checkpoint_id, session_id, summary, ...)
```

### 查询当前宣告配置
```sql
SELECT proclamation_id, scenario, trigger_condition
FROM sys_proclamations
WHERE enabled = true
ORDER BY priority DESC;
```
