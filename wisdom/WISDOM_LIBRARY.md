### 无为而治

```
无为 ≠ 不作为
无为 = 顺势而为，不强求

Solar 映射:
• 任务简单 → 用简单方案 (不过度设计)
• 已有方案好用 → 复用 (不重新发明)
• 遇到阻力 → 换个角度 (不硬碰硬)
• 时机未到 → 等待 (不急躁)
```

# 第四部分：系统表设计 (IaST)

```sql
-- 智慧库表
CREATE TABLE IF NOT EXISTS sys_wisdom (
    id INTEGER PRIMARY KEY,
    tradition TEXT NOT NULL,          -- 'eastern', 'western'
    school TEXT NOT NULL,             -- '儒家', 'stoicism', etc.
    thinker TEXT,                     -- 孔子, Aristotle, etc.
    concept TEXT NOT NULL,            -- 核心概念
    original_text TEXT,               -- 原文
    translation TEXT,                 -- 翻译/解释
    solar_application TEXT,           -- Solar 应用
    keywords TEXT,                    -- JSON 关键词列表
    embedding BLOB,                   -- 向量嵌入 (用于语义搜索)
    created_at TEXT DEFAULT CURRENT_TIMESTAMP
);

-- 智慧应用日志
CREATE TABLE IF NOT EXISTS sys_wisdom_application_log (
    id INTEGER PRIMARY KEY,
    wisdom_id INTEGER,
    task_context TEXT,                -- 任务上下文
    how_applied TEXT,                 -- 如何应用
    outcome TEXT,                     -- 结果
    was_helpful BOOLEAN,              -- 是否有帮助
    created_at TEXT DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (wisdom_id) REFERENCES sys_wisdom(id)
);

-- 智慧搜索视图
CREATE VIEW v_wisdom_by_keyword AS
SELECT
    tradition,
    school,
    thinker,
    concept,
    original_text,
    solar_application
FROM sys_wisdom
ORDER BY school, thinker;
```
