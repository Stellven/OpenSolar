# Demo 2: 智能代码审计

> 触发词: `审计 ThunderDuck 项目的内存安全，生成报告`

```

```

## 亮点

1. **Agent 动态组合** - 根据任务特点自动选择和增强 Agent
2. **专业级分析** - 12 条内存安全检查规则
3. **实时进度** - TVS 展示扫描过程
4. **专业报告** - 结构化、可操作的审计报告
5. **知识沉淀** - 审计规则和结果入库，持续学习
6. **闭环操作** - 提供具体的修复建议和后续操作
