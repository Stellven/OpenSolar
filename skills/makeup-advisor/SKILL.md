# AI 美妆顾问

> 拍照 → 面部分析 → 结合趋势 → 生成个性化化妆建议

## 功能

1. **面部特征分析**
   - 脸型检测（圆脸/方脸/鹅蛋脸/心形脸/长脸）
   - 五官比例（眼距、鼻长、唇厚等）
   - 肤色分析（冷暖色调）
   - 肤质判断（干性/油性/混合性）

2. **趋势感知**
   - 当前节假日（情人节/春节/圣诞等）
   - 季节流行色
   - 明星热门妆容
   - 小红书/抖音趋势

3. **个性化建议**
   - 适合的妆容风格
   - 具体化妆步骤
   - 产品推荐（平价/高端）
   - 避雷指南

## 使用

```bash
/makeup-advisor           # 拍照分析
/makeup-advisor --trend   # 查看当前流行趋势
/makeup-advisor --date valentine  # 情人节妆容建议
```

## 技术栈

- MediaPipe Face Mesh - 468个面部关键点
- OpenCV - 图像处理
- 多模态大模型 - 妆容建议生成
