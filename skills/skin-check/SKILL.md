# /skin-check - 智能皮肤检测

## 功能

通过摄像头拍照，使用 AI 视觉模型分析皮肤状况，并由四位专家（Gemini Pro, DeepSeek R1, GLM-5, 稳健派）进行多维度评审，给出专业建议。

## 执行步骤

### 1. 拍照
- 调用 selfie skill 拍摄皮肤照片
- 支持倒计时，方便调整角度

### 2. 初步分析
- 使用 Gemini 2.0 Flash 进行视觉分析
- 识别：皮肤类型、问题区域、整体状况

### 3. 专家评审（四大老专家）
并行调用四位专家进行深度分析：
- **Gemini 2.5 Pro (稳健派)**: 严谨的医学角度分析
- **DeepSeek R1 (审判官)**: 深度推理，找出潜在问题
- **GLM-5 (智囊)**: 中文友好的生活建议
- **Gemini 3 Pro (探索派)**: 综合评估与权衡

每位专家关注点：
- 皮肤问题识别（痘痘、斑点、细纹等）
- 皮肤类型判断（油性/干性/混合性）
- 护理建议（清洁、保湿、防晒）
- 就医建议（严重问题需要看医生）

### 4. 综合报告
整合四位专家意见，生成结构化报告：
- 一致性建议（所有专家都同意的）
- 分歧点（专家意见不一致的地方）
- 优先级建议（最需要关注的问题）

## 使用方式

```bash
# 基础检测
/skin-check

# 指定关注点
/skin-check 痘痘
/skin-check 斑点
/skin-check 整体
```

## 技术实现

### Phase 1 (当前)
- 拍照: selfie skill
- 视觉分析: Gemini 2.0 Flash
- 专家评审: brain-router 并行调用 4 个模型

### Phase 2 (未来)
- 本地模型: CoreML (MobileNet/EfficientNet)
- 病灶检测: YOLOv8
- 历史对比: 记录检测历史，跟踪变化

## 配置要求

需要设置以下环境变量（至少 GOOGLE_API_KEY）：

```bash
export GOOGLE_API_KEY="your-google-api-key"      # Gemini (必需)
export DEEPSEEK_API_KEY="your-deepseek-api-key"  # DeepSeek (可选)
export GLM_API_KEY="your-glm-api-key"            # GLM (可选)
```

**说明:**
- 如果只设置了 `GOOGLE_API_KEY`，其他专家会使用模拟回复
- 建议至少配置 2-3 个 API key 以获得多专家真实评审

## 注意事项

1. **免责声明**: 此检测仅供参考，不能替代专业医疗诊断
2. **隐私保护**: 照片仅用于分析，不会上传或保存
3. **光线要求**: 建议在自然光下拍摄，效果更好
4. **角度建议**: 正面45度角拍摄效果最佳
