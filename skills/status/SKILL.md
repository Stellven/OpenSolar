# /status - Solar 系统状态

## 功能

显示太阳系风格的系统状态图，展示当前活跃 Agent 和调用关系。

|        |                   |                   |           |
|        v                   v                   v           |
|  +-----------+      +-----------+      +-----------+       |
|  | Researcher| ---> | Architect | ---> |   Coder   |       |
|  |  (Opus)   |      |  (Opus)   |      | (Sonnet)  |       |

## 简化视图 (每次操作后自动显示)

```

```

## 触发条件

- `/status` - 显示完整太阳系视图
- `/status mini` - 显示简化状态栏
- 自动显示 - 每次阶段切换时
