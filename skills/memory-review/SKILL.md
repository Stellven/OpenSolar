# /memory-review - 记忆回顾与总结

> 记忆是智慧的源泉，总结是成长的阶梯

## 用法

```bash
/memory-review                # 显示记忆状态概览
/memory-review recent         # 查看最近记忆
/memory-review search <query> # 搜索记忆
/memory-review summarize      # 总结并压缩记忆
/memory-review cleanup        # 清理低价值记忆
/memory-review stats          # 记忆统计
```

## 记忆总结流程

```
碎片记忆 (100 条)
       │
       ▼
   按主题聚类
       │
       ▼
   提取共性模式
       │
       ▼
   生成总结 (10 条结构化知识)
       │
       ▼
   验证准确性
       │
       ▼
   存入结构化知识
       │
       ▼
   原始碎片降级或删除
```

## 总结策略

### 1. 主题聚合

```sql
-- 按主题聚合相似记忆
SELECT namespace, COUNT(*) as count
FROM evo_memory_semantic
WHERE source_type = 'learned'
  AND created_at > datetime('now', '-7 days')
GROUP BY namespace
ORDER BY count DESC;
```

### 2. 提取要点

```typescript
// 对同一主题的多条记忆进行总结
const summary = await summarize({
  memories: relatedMemories,
  strategy: 'extract_key_points',
  maxPoints: 5
});
```

### 3. 压缩存储

```sql
-- 插入总结记忆
INSERT INTO evo_memory_semantic (
    memory_id,
    namespace,
    key,
    value,
    source_type,
    confidence
) VALUES (
    'summary_xxx',
    'solar_knowledge/ai/agent_design',
    'Agent 设计经验总结 (2026-01)',
    '{
      "summary": "本月学习的 Agent 设计要点总结...",
      "source_count": 15,
      "key_insights": [...],
      "created_from": ["learn_001", "learn_002", ...]
    }',
    'summarized',
    0.9
);

-- 降级原始记忆
UPDATE evo_memory_semantic
SET confidence = confidence * 0.5
WHERE memory_id IN ('learn_001', 'learn_002', ...);
```

```
• 不复习的记忆会逐渐遗忘 (置信度下降)

```
