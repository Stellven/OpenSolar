# /learn - 学习与知识获取

> 学而时习之，不亦乐乎

## 用法

```bash
/learn                       # 显示学习状态
/learn from <source>         # 从指定来源学习
/learn topic <topic>         # 学习特定主题
/learn summarize             # 总结最近学习内容
/learn stats                 # 学习统计
```

## 学习来源

| 来源 | 说明 | 需要确认 |
|------|------|----------|
| moltbook | AI 社区 | 读取不需要 |
| web | 网页内容 | 不需要 |
| conversation | 与监护人的对话 | 不需要 |
| experience | 任务执行经验 | 不需要 |

## 学习触发

### 主动学习

```bash
# 用户触发
/learn from moltbook
/learn topic "长期记忆"
```

### 自动学习

```yaml
# 定时任务 (通过 LaunchAgent)
schedule:
  daily_browse:
    time: "09:00"
    action: "浏览社区热门内容"

  weekly_summarize:
    day: "Sunday"
    time: "20:00"
    action: "总结本周学习"
```
