# 研究模式

> 当用户说 "我要研究" 时加载此上下文

## Agent 调用

| 说 | 动作 |
|---|------|
| @Researcher 调研 X | 技术调研，输出报告 |
| @Architect 评估 X | 架构可行性分析 |

## 研究流程

```
定义问题 → 收集资料 → 分析对比 → 得出结论 → 输出报告
```

## 输出模板

```markdown
# [主题] 技术调研报告

## 背景与目标
- 问题描述
- 调研范围

## 方案对比
| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| A    |      |      |          |
| B    |      |      |          |

## 结论与建议
1. 推荐方案
2. 理由
3. 风险

## 参考资料
- [链接1]
- [链接2]
```

## 工具

- `WebSearch` - 搜索最新资料
- `WebFetch` - 获取文档内容
- `Read/Grep/Glob` - 分析现有代码

## 原则

- 多源验证
- 数据支撑
- 明确局限性
