# 📚 Solar CEO 图书馆

> 董事长放书，CEO 读书学习

## 书架分类

| 目录 | 放什么 | 示例 |
|------|--------|------|
| `theory/` | 论文、研究报告、理论分析 | PDF论文、学术文章 |
| `practice/` | 项目经验、优化方案、实验结论 | 技术博客、案例分析 |
| `wisdom/` | 经典智慧、商业法则、管理经验 | 商业书籍、名人传记 |
| `reference/` | 技术文档、API参考、最佳实践 | 官方文档、手册 |
| `lessons/` | 踩坑记录、失败经验、避坑指南 | 事故报告、教训总结 |

## 支持格式

- `.md` Markdown
- `.pdf` PDF 文档
- `.txt` 纯文本
- `.json` 结构化数据
- `.html` 网页存档

## 使用方式

放入文件后，Solar 会自动：
1. 扫描新书
2. 提取摘要
3. 入库索引
4. 可随时查阅
