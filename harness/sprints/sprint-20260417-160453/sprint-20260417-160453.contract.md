## 需求

Solar Subconscious (自研版): 让 harness 三化身自动从 Sprint 经验中学习，零外部依赖。

**核心闭环**:
```
Claude 完成一轮 Sprint 响应
  → Stop hook 异步提取 1-3 条教训 (brain-router 调小模型)
  → 写入 lessons.jsonl (持久)
  → 下次 UserPromptSubmit hook whisper 相关教训
  → Claude 避坑
  → 更好的执行 → 更好的教训 → ...
```

**用到的自有组件** (零外部依赖):
- hooks: Stop + UserPromptSubmit
- brain-router: 调 glm-4-flash 提取教训 (成本 ~$0.0001/次)
- MemPalace: 语义搜索找相关教训 (可选加速)
- events.jsonl: Sprint 完整轨迹
- eval.json: 结构化反馈 (FAIL 原因)

## 设计要点

### D1: subconscious-learn.sh

```bash
#!/bin/bash
# Stop hook — 异步提取教训，不阻塞 Claude

BRAIN_DIR="$HOME/.solar/harness/brain"
SPRINTS_DIR="$HOME/.solar/harness/sprints"

# 找最近完成的 Sprint
LATEST_EVAL=$(ls -t "$SPRINTS_DIR"/*.eval.json 2>/dev/null | head -1)
[[ -z "$LATEST_EVAL" ]] && exit 0

SID=$(basename "$LATEST_EVAL" .eval.json)

# 已学过? 幂等检查
grep -q "\"sprint_id\":\"$SID\"" "$BRAIN_DIR/lessons.jsonl" 2>/dev/null && exit 0

# 后台异步提取 (不阻塞 Claude)
(
  EVAL=$(cat "$LATEST_EVAL")
  HANDOFF=$(cat "$SPRINTS_DIR/${SID}.handoff.md" 2>/dev/null | head -50)

  # 调 brain-router 提取教训
  LESSONS=$(mcp__brain-router__complete \
    model="glm-4-flash" \
    system="你是经验提取器。从 Sprint 评审结果中提取 1-3 条可复用教训。每条 < 80 字。输出 JSON 数组。" \
    prompt="eval.json: $EVAL\n\nhandoff (前50行): $HANDOFF")

  # 写入 lessons.jsonl
  echo "$LESSONS" | python3 -c "
import json, sys, datetime
data = json.loads(sys.stdin.read())
ts = datetime.datetime.utcnow().strftime('%Y-%m-%dT%H:%M:%SZ')
sid = '$SID'
for item in (data if isinstance(data, list) else [data]):
    lesson = item if isinstance(item, str) else item.get('lesson', str(item))
    print(json.dumps({
        'ts': ts, 'sprint_id': sid, 'lesson': lesson,
        'source': 'eval', 'confidence': 0.8, 'tags': []
    }, ensure_ascii=False))
" >> "$BRAIN_DIR/lessons.jsonl"
) &
```

### D2: subconscious-whisper.sh

```bash
#!/bin/bash
# UserPromptSubmit hook — 注入相关教训，< 2 秒

BRAIN_DIR="$HOME/.solar/harness/brain"
LESSONS_FILE="$BRAIN_DIR/lessons.jsonl"

[[ ! -f "$LESSONS_FILE" ]] && exit 0

# 读最近 20 条教训
RECENT=$(tail -20 "$LESSONS_FILE")

# 简单关键词匹配当前 prompt
PROMPT="${CLAUDE_USER_PROMPT:-}"
[[ -z "$PROMPT" ]] && exit 0

# 匹配 (用 grep 极快)
MATCHED=$(echo "$RECENT" | while IFS= read -r line; do
  lesson=$(echo "$line" | python3 -c "import json,sys; print(json.loads(sys.stdin.read()).get('lesson',''))" 2>/dev/null)
  # 提取教训中的关键词，看 prompt 里有没有
  for word in $(echo "$lesson" | tr ' /' '\n' | grep -E '.{3,}' | head -5); do
    if echo "$PROMPT" | grep -qi "$word" 2>/dev/null; then
      echo "$lesson"
      break
    fi
  done
done | head -3)

[[ -z "$MATCHED" ]] && exit 0

# 注入 whisper
echo "<system-reminder>"
echo "Solar Subconscious 经验提醒:"
echo "$MATCHED" | while IFS= read -r m; do
  echo "- ⚡ $m"
done
echo "</system-reminder>"
```
