## Design / жҠҖжңҜж–№жЎҲ

Build a deterministic local v1 with three layers:

1. Extract terminal sprint evidence into compact workflow/failure/repair records.
2. Query records by state signature, trigger signature, tags, outcome, and usage stats.
3. Inject bounded observe-only coordinator advisory before dispatch; fail open on any error.
