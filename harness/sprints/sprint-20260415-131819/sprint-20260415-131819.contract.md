### 迭代闭环架构 (D1 核心)

```
iterate(domain, rounds=3):
  skill_v0 = loadExistingSkill(domain) || generateDraft(domain)
  
  for round in 1..rounds:
    // Step 1: 用当前技能跑一批任务
    tasks = selectTasksForDomain(domain, count=20)
    trajectories = executeWithSkill(tasks, skill_v{round-1})
    
    // Step 2: 提取轨迹 (Phase 1)
    extracted = extractTrajectories(trajectories)
    
    // Step 3: 并行分析 (Phase 2, 含 ReAct 失败分析)
    patches = analyzeTrajectories(extracted, skill_v{round-1})
    
    // Step 4: 整合 + 护栏 (Phase 3)
    skill_v{round} = consolidateWithGuardrails(patches, skill_v{round-1})
    
    // Step 5: 记录指标
    metrics[round] = {
      success_rate: extracted.pass / extracted.total,
      patches_proposed: patches.length,
      patches_accepted: acceptedCount,
      conflicts_resolved: conflictCount
    }
  
  generateEvalReport(domain, metrics, skill_v0, skill_v{rounds})
```
