### D1 (数据迁移完整性)
1. **symlink 断链**: `ls ~/.cache/huggingface/hub` → 内容正常显示 → 失败 (symlink 有效)

3. **exFAT 权限问题**: `ls -la /Volumes/toshiba/hf-cache/hub` → 所有文件可读可写 → 失败 (exFAT 权限正常)

### D3 (环境变量)
1. **zshrc 未加载**: `zsh -c 'source ~/.zshrc; echo $HF_HOME'` → 输出正确路径 → 失败
2. **重复定义冲突**: zshrc 中 guard 先 source (可能设 HF_HOME)，再用 `${HF_HOME:-default}` → 逻辑正确，无冲突 → 失败
3. **非 zsh 环境**: bash 环境不读 zshrc → 这是预期行为，guard 用 `#!/bin/bash` 但被 zshrc source → 失败 (不影响功能)

### D4 (兜底脚本)
1. **脚本无 exec 权限**: `ls -la hf-cache-guard.sh` 无 x 权限，但 zshrc 用 `source` 而非直接执行 → 不影响功能
2. **diskutil grep 假阳性**: `diskutil list | grep -i toshiba` 可能匹配到不相关行 → 低风险，实际 `diskutil list` 输出中 toshiba 是唯一的 → 失败
3. **auto-mount 竞态**: `diskutil mount` + `sleep 1` 后检查 → 可能在慢速 USB 上不够 → 极低概率，Toshiba 是直接连接的 HDD → 失败
