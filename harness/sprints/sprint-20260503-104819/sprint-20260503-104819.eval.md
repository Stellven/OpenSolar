## 额外发现

### 1. D4 合约偏离 (合理)
合约要求 `flock -n/flock -u`, 实际用 `mkdir` 原子锁。macOS 无 flock, 偏离合理。但 mkdir 锁的 stale 风险 (进程 crash 后锁残留) 与 flock 不同 (flock 进程死自动释放)。D2 的 30min 超时缓解此风险。

### 2. D8 test-multi-sprint-race.sh 预存 bug
`((total++))` 在 bash set -e 下 total=0 时导致脚本退出。修复: `total=$((total+1))` 或 `((total++)) || true`。建议下个 sprint 修复。

### 3. test-dispatch-modal.sh FAIL 非功能性
FAIL 项 "coordinator.sh 未找到 D3 预解锁代码" 和 "未找到派发后验证代码" 可能是 grep pattern 未更新 (代码行号变动)。不影响功能正确性。

### 4. rollback_state_cache python3 静默失败
handoff 已标注 `|| true`, 不阻塞主流程, 但如果 python3 失败, state cache 不会被清理, 下轮可能不触发 dispatch。低风险。
