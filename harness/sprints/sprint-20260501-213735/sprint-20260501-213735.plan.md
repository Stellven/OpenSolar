### D4: update-contract 诊断

**源码位置**: `~/.solar/harness/solar-harness.sh:915-943`

**regex 分析**:
```python
pattern = r'(## Done 定义\n\n>.*?\n\n)(.*?)(\n## )'
```

**问题**: 合约中实际的 Done 段格式是:
```markdown
## Definition of Done

- [ ] D1: (criterion)
```

而 regex 期望:
```markdown
## Done 定义\n\n>.*?\n\n<content>\n\n## 
```

两个不匹配:
1. 标题是 `## Definition of Done` 不是 `## Done 定义`
2. 标题后直接 `- [ ]` 没有 `> Planner fills in` 引用块

→ regex 匹配不到，`re.sub` 不替换，文件内容不变。

**修复建议**: 改 regex 为匹配两种格式，或统一合约模板。
