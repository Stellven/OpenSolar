# Execution Contract — Toy Mac Mini Smoke Contract

## Intent

Verify that a contract written on the laptop can be delivered to the Mac mini Solar contract intake and executed safely by the remote Solar harness.

This is a no-op smoke contract. It must not modify project source code, Solar DB, knowledge files, or runtime configuration.

## Scope

Allowed:
- Read basic host facts.
- Read Solar harness directory presence.
- Write one proof handoff file under `~/.solar/harness/sprints/`.

Forbidden:
- Do not edit files outside `~/.solar/harness/sprints/`.
- Do not delete, move, archive, or rsync anything.
- Do not start long-running services.
- Do not change Codex, Solar, solar-harness, knowledge base, or shell profile configuration.

## Required Work

1. Confirm this contract is running on the Mac mini account.
2. Confirm these paths exist:
   - `~/.solar/harness`
   - `~/.solar/harness/sprints`
   - `~/.solar/codex-bridge/from-codex`
3. Write a handoff file:
   - `~/.solar/harness/sprints/sprint-toy-mini-smoke.handoff.md`

## Acceptance Criteria

### A1 — Host Smoke

The handoff includes:
- `hostname`
- `whoami`
- `pwd`
- timestamp in UTC

Verify:

```bash
test -f "$HOME/.solar/harness/sprints/sprint-toy-mini-smoke.handoff.md"
grep -q "hostname:" "$HOME/.solar/harness/sprints/sprint-toy-mini-smoke.handoff.md"
grep -q "whoami:" "$HOME/.solar/harness/sprints/sprint-toy-mini-smoke.handoff.md"
```

### A2 — Harness Pool Exists

The handoff includes `contract_pool: ok` only if both directories exist:

```bash
test -d "$HOME/.solar/harness/sprints"
test -d "$HOME/.solar/codex-bridge/from-codex"
```

### A3 — No Source Changes

The handoff states `source_changes: none`.

## Expected Handoff Format

```markdown
# Handoff — sprint-toy-mini-smoke

- status: passed
- hostname: <hostname>
- whoami: <user>
- cwd: <pwd>
- utc: <timestamp>
- contract_pool: ok
- source_changes: none
- notes: toy contract executed safely
```

## Stop Rules

- Stop immediately if the task would require changing source code or configuration.
- Stop immediately if `~/.solar/harness/sprints` is missing.
- Do not attempt repair in this toy contract.
