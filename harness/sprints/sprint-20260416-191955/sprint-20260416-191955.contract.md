```
     → Phase 2: C1-C7 自动检测
     → Phase 3: Q1-Q5 诛心五问
     → Phase 4: READY / NOT READY
  2. 嵌入技能输出到 eval.md
  3. 根据 READY/NOT_READY 决定 verdict
  4. 同时输出 eval.json（含 verify_all_invoked: true）

时间: 2 分钟
质量: 标准化 + 诛心五问终审
```

### /verify-all 技能的 12 项检查

**Phase 2 七项自动检测 (C1-C7):**
- C1 功能完备 (无 TODO/FIXME/stub)
- C2 无断头 (有入口调用方)
- C3 自动触发 (hook/crontab/import 链)
- C4 默认使用 (无需额外配置)
- C5 激活口令 (intent-engine 注册)
- C6 输出持久化
- C7 与其他模块集成

**Phase 3 诛心五问 (Q1-Q5):**
- Q1 你真的在跑吗？（不是只写了代码）
- Q2 你真的有效吗？（产出有价值）
- Q3 你真的会退化吗？（覆盖率/质量能追踪）
- Q4 你真的能恢复吗？（崩溃后可修复）
- Q5 你真的用了吗？（有调用记录）
