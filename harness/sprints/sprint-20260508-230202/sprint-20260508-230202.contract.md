# Execution Contract — Toy Mac Mini Auto Smoke Contract

## Intent

Verify the full remote path: laptop writes a contract, Mac mini chain-watcher ingests it, Mac mini coordinator routes it to builder, and builder produces the handoff.

This contract is a no-op smoke test. It must not modify project source, Solar DB, knowledge files, Codex config, shell profiles, or harness runtime config.

## Scope

Allowed:
- Read basic host facts.
- Read directory existence.
- Write only the sprint handoff file for the actual generated sprint id.
- Submit the handoff using `solar-harness.sh handoff-submit`.

Forbidden:
- Do not edit files outside `~/.solar/harness/sprints/`.
- Do not delete, move, archive, or rsync anything.
- Do not start long-running services.
- Do not change Solar, solar-harness, Codex, knowledge base, or shell profile configuration.

## Required Work

1. Read this contract from `~/.solar/harness/sprints/<actual-sid>.contract.md`.
2. Derive `SID` from the contract filename, for example:
   - `SID=$(basename "$CONTRACT" .contract.md)`
3. Confirm these paths exist:
   - `~/.solar/harness`
   - `~/.solar/harness/sprints`
   - `~/.solar/codex-bridge/from-codex`
4. Write the handoff file:
   - `~/.solar/harness/sprints/${SID}.handoff.md`
5. Run the verification commands below.
6. Submit:
   - `bash ~/.solar/harness/solar-harness.sh handoff-submit "$SID"`

## Acceptance Criteria

### A1 — Host Smoke

The handoff includes:
- `hostname:`
- `whoami:`
- `cwd:`
- `utc:`

Verify:

```bash
test -f "$HOME/.solar/harness/sprints/${SID}.handoff.md"
grep -q "hostname:" "$HOME/.solar/harness/sprints/${SID}.handoff.md"
grep -q "whoami:" "$HOME/.solar/harness/sprints/${SID}.handoff.md"
grep -q "cwd:" "$HOME/.solar/harness/sprints/${SID}.handoff.md"
grep -q "utc:" "$HOME/.solar/harness/sprints/${SID}.handoff.md"
```

### A2 — Contract Pool Exists

The handoff includes `contract_pool: ok` only if both directories exist:

```bash
test -d "$HOME/.solar/harness/sprints"
test -d "$HOME/.solar/codex-bridge/from-codex"
```

### A3 — No Source Changes

The handoff states:

```text
source_changes: none
```

## Expected Handoff Format

```markdown

```
