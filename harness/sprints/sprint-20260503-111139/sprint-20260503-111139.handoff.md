## D4 修复详情
```bash
# Before (flock, macOS 不可用):
PID_FILE="$HOME/.solar/harness/.chain-watcher.pid"
exec 200>"$PID_FILE"
flock -n 200 || { echo "[chain-watcher] already running, exit"; exit 0; }
echo $$ > "$PID_FILE"

# After (mkdir 原子锁, macOS 原生):
PID_LOCK_DIR="$HOME/.solar/harness/.chain-watcher.lock"
mkdir "$PID_LOCK_DIR" 2>/dev/null || { echo "[chain-watcher] already running, exit"; exit 0; }
echo $$ > "$PID_LOCK_DIR/pid"
trap 'rm -rf "$PID_LOCK_DIR"' EXIT
```
