# Plan — GEPA-Optimized Self-Evolving Requirement Compiler

> **Sprint**: `sprint-20260525-000730`
> **Slice**: `strategy` (P0/P1)

## Goal

把 Requirement Compiler 升级为 GEPA 离线可优化的自进化系统：compiler_profile registry + CompileEvalHarness + ASI trace + GEPA adapter + hard validators。

## Parallelization

```text
P1 (profile schema)     ─┐
P2 (profile registry)   ─┤ P1→P2 串行
P3 (eval dimensions)    ─┤ 可与 P2 并行
P4 (hard validators)    ─┤ 可与 P2/P3 并行
P5 (ASI trace)          ─┤ 依赖 P3
P6 (golden cases)       ─┤ 可与 P2 并行
P7 (GEPA adapter)       ─┤ 依赖 P2+P3+P5
P8 (integration tests)  ─┘ 依赖全部
```

## Execution Plan

### P1: compiler_profile Schema
- `lib/compiler_profile/schema.py` — Profile JSON schema 定义
- 6 类 policy schema (intake/ir/contract/dag/evidence/handoff)
- **verify**: schema validation against test profile

### P2: compiler_profile Registry
- `lib/compiler_profile/registry.py` — register/query/activate/list/history
- `lib/compiler_profile/loader.py` — JSON files + SQLite cache
- **verify**: register + query + activate roundtrip

### P3: CompileEvalHarness Dimensions
- `lib/compile_eval/dimensions.py` — 7 评估维度实现
- **verify**: 对 mock artifacts 运行 → score 输出

### P4: Hard Validators
- `lib/compile_eval/hard_validators.py` — 6 个 hard validator
- **verify**: 注入违规用例 → ASI=0

### P5: ASI Trace
- `lib/compile_eval/asi_trace.py` — trace schema + SQLite storage
- **verify**: 写入 + 查询 roundtrip

### P6: Golden Cases
- `lib/compile_eval/golden_cases.py` — 从 accepted artifacts 加载
- **verify**: 至少 3 个 golden cases 加载成功

### P7: GEPA Adapter
- `lib/compile_eval/harness.py` — main eval harness class
- 复用 `integrations/gepa_optimizer/` 的 run/propose/promote/rollback
- **verify**: fitness_function 对 golden cases 运行

### P8: Integration Tests
- 全链路测试: profile → compile → eval → ASI trace → GEPA propose
- 回归: 现有编译管线不受影响
- **verify**: AC1-AC10 全部可复现

## Rollback / Stop Rules

- rollback: 删除 `lib/compiler_profile/` 和 `lib/compile_eval/`; 恢复 `integrations/gepa_optimizer/` 的 extend 改动
- stop: 任一 hard validator 未实现 → 不算 done
- stop: GEPA adapter 未接入现有 run/propose → 不算 done
- stop: 新建平行 optimizer substrate → 违规
