1. GPU SIMD Reduce (Q1, Q6)
利用 Metal simd_sum/simd_ballot 进行 32 线程并行归约，避免原子操作瓶颈
// SIMD Group 归约 (32→1)
float sum = simd_sum(value);

// 仅 lane 0 执行原子加
if (simd_lane == 0) {
    atomic_fetch_add_explicit(result, sum, memory_order_relaxed);
}

for (size_t i = 0; i < ord.count; ++i) {
    int32_t ck = ord.o_custkey[i];
    has_order_bitmap[ck >> 6] |= (1ULL << (ck & 63));
}

3. 融合 Filter + Aggregate (Q1, Q6)
单 Kernel 完成过滤和聚合，避免中间结果物化
kernel void fused_filter_agg(...) {
    // 向量化谓词评估
    bool pass = (shipdate[gid] >= params.date_min) &
                (shipdate[gid] < params.date_max) &
                (discount[gid] >= params.disc_min);

    // 条件计算
    float value = pass ? (extprice[gid] * discount[gid]) : 0.0f;

    // SIMD 归约 + 原子累加
    float sum = simd_sum(value);
    if (simd_lane == 0) atomic_fetch_add(result, sum);
}
📈 减少 50% 内存带宽，提升 2-3x
4. GPU 列式零拷贝存储
MTLResourceStorageModeShared 实现 CPU/GPU 共享内存，无需数据传输

                                  options:MTLResourceStorageModeShared];
data_ = static_cast([mtl_buffer_ contents]);
