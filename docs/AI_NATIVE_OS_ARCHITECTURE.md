## 架构层次说明

### Layer 1: Human-Machine Interface (人机交互层)
- **TUV Engine**: 终端 UI 框架，基于 TVS/VDL
- **Solar UI Engine**: 可视化仪表盘，实时监控

### Layer 2: Orchestration (编排调度层)
- **Solar Flow Controller**: 五阶段流程引擎
- **State Manager**: 项目状态持久化
- **Context Router**: 模式检测与懒加载
- **Adaptive Intelligence Engine** (NEW): 用户行为分析与自适应推荐
  - Behavior Recorder: 记录用户所有操作行为
  - Preference Analyzer: 分析行为模式，提取用户偏好
  - Adaptive Recommender: 基于偏好推荐 Skills/Agents/Hooks/Workflows

### Layer 3: Execution Framework (执行框架层)
- **Agent System**: 13 个专业 Agent 协作
- **Hooks**: 事件驱动的扩展机制
- **Skills**: 可复用的技能模块
- **MCP**: 标准化工具协议

### Layer 4: AI Engine (AI 驱动层)
- **Model Router**: 智能模型选择
- **Multi-Model**: Opus/Sonnet/Haiku + 第三方模型

### Layer 5: Resource (资源抽象层)
- 文件、网络、数据库、系统调用统一抽象
- **Apple Platform Integration** (NEW): macOS/iOS 深度集成
  - Shortcuts Manager: 管理和运行 Apple 快捷指令
  - Siri Integration: 语音触发和 Siri 建议
  - Automation Triggers: 时间/位置/App/事件触发器
  - Solar-Shortcuts Bridge: Solar 命令与快捷指令双向映射
