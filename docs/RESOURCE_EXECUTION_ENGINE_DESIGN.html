📋 系统表设计 (IaST)

sys_scripts - 脚本注册表

字段 类型 说明
script_id TEXT PK 唯一标识 (hash)
name TEXT 脚本名称
description TEXT 功能描述 (用于匹配)
intent_keywords JSON 意图关键词列表
runtime TEXT bun/node/python/bash
code TEXT 脚本代码
params_schema JSON 参数定义
example_inputs JSON 示例输入
version INTEGER 版本号
status TEXT active/deprecated/archived
created_at DATETIME 创建时间

sys_script_stats - 使用统计

字段 类型 说明
script_id TEXT FK 关联脚本
invoke_count INTEGER 调用次数
success_count INTEGER 成功次数
avg_latency_ms REAL 平均延迟
last_used_at DATETIME 最后使用时间
hot_score REAL 热度分数
-- 热度计算公式 UPDATE sys_script_stats SET hot_score = invoke_count * 0.4 + (success_count/invoke_count) * 0.3 + (1.0 / (julianday('now') - julianday(last_used_at) + 1)) * 0.3;

sys_script_embeddings - 语义索引

字段 类型 说明
script_id TEXT FK 关联脚本
embedding BLOB 向量嵌入 (1536d)
text_hash TEXT 原文哈希 (变更检测)
-- 语义匹配查询 SELECT s.*, cosine_similarity(e.embedding, ?) AS sim FROM sys_scripts s JOIN sys_script_embeddings e ON s.script_id = e.script_id WHERE sim > 0.8 ORDER BY sim DESC LIMIT 5;

sys_resource_priority - 优先级配置

字段 类型 说明
resource_type TEXT PK shortcut/script/skill/mcp/agent/code
priority INTEGER 优先级 (1-6)
enabled BOOLEAN 是否启用
match_threshold REAL 匹配阈值
max_latency_ms INTEGER 超时阈值
-- 默认优先级配置 INSERT INTO sys_resource_priority VALUES ('shortcut', 1, true, 0.7, 100), ('script', 2, true, 0.8, 200), ('skill', 3, true, 0.7, 500), ('mcp', 4, true, 0.6, 1000), ('agent', 5, true, 0.5, 5000), ('code', 6, true, 0.0, 10000);

🔌 API 设计

ResourceMatcher API

interface ResourceMatcher { // 匹配最佳资源 match(intent: Intent): Promise<MatchResult>; // 按优先级逐层匹配 matchByPriority( intent: Intent, levels: PriorityLevel[] ): Promise<MatchResult | null>; // 执行匹配到的资源 execute( match: MatchResult, params: Record<string, any> ): Promise<ExecutionResult>; } type PriorityLevel = | 'shortcut' // P1 | 'script' // P2 | 'skill' // P3 | 'mcp' // P4 | 'agent' // P5 | 'code'; // P6

ScriptEngine API

interface ScriptEngine { // 注册新脚本 register(script: ScriptDefinition): Promise<string>; // 语义搜索脚本 search( query: string, options?: { threshold?: number; // 默认 0.8 limit?: number; // 默认 5 } ): Promise<ScriptMatch[]>; // 执行脚本 execute( scriptId: string, params: Record<string, any> ): Promise<ExecutionResult>; // 更新统计 recordStats( scriptId: string, success: boolean, latencyMs: number ): Promise<void>; // 清理过期脚本 cleanup(daysUnused?: number): Promise<number>; }
