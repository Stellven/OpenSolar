    15.2 技术一: Bitmap 替代 Hash Set
          ┌─────────────────────────────────────────────────────────┐
          │              Bitmap vs Hash Set                          │
          ├─────────────────────────────────────────────────────────┤
          │                                                         │
          │  // ❌ Hash Set: O(1) 但常数大                          │
          │  std::unordered_set<int32_t> german_suppliers;          │
          │  // 问题: 哈希计算 + 内存随机访问 + 分支预测失败         │
          │                                                         │
          │  // ✅ Bitmap: O(1) 且缓存友好                          │
          │  std::vector<uint8_t> supp_bitmap((max_key + 8) >> 3);  │
          │                                                         │
          │  // Set bit                                             │
          │  supp_bitmap[key >> 3] |= (1u << (key & 7));            │
          │                                                         │
          │  // Check bit                                           │
          │  bool exists = supp_bitmap[key >> 3] & (1u << (key & 7));│
          │                                                         │
          │  优势:                                                   │
          │  • 无哈希计算                                            │
          │  • 连续内存访问                                          │
          │  • 缓存友好                                              │
          │  • 无分支预测问题                                        │
          │                                                         │
          └─────────────────────────────────────────────────────────┘
    15.3 技术二: Dense Array 替代 Hash Map
          ┌─────────────────────────────────────────────────────────┐
          │              Dense Array vs Hash Map                     │
          ├─────────────────────────────────────────────────────────┤
          │                                                         │
          │  // ❌ Hash Map                                          │
          │  std::unordered_map<int32_t, int64_t> part_values;      │
          │  part_values[partkey] += value;  // 哈希 + 查找 + 插入  │
          │                                                         │
          │  // ✅ Dense Array                                       │
          │  std::vector<int64_t> part_values(max_partkey + 1, 0);  │
          │  part_values[partkey] += value;  // 直接寻址，O(1)      │
          │                                                         │
          │  适用条件:                                               │
          │  • Key 范围有限且已知                                    │
          │  • Key 分布相对密集                                      │
          │  • 内存开销可接受                                        │
          │                                                         │
          └─────────────────────────────────────────────────────────┘
    15.4 技术三: 单线程避免同步开销
