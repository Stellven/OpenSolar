# 洞察报告: Agent 记忆机制论文分析

> **论文**: Rethinking Memory Mechanisms of Foundation Agents in the Second Half: A Survey
> **日期**: 2026-02-11
> **分析模式**: /insight 七阶段流程
> **参与专家**: 千里马(Gemini 3 Pro), 思考驼(DeepSeek R1), 老实人(GLM-4-Plus)

## 一、论文概述

这是一篇关于基础模型 Agent 记忆机制的系统性综述，涵盖 218 篇相关论文，提出了一个 3D 分类框架来组织和理解当前 Agent 记忆系统的发展。

### 论文核心问题

1. 现有 Agent 记忆系统如何分类？
2. 记忆的基本操作有哪些？
3. 多 Agent 场景下记忆如何共享与协作？
4. 未来研究方向是什么？

### 操作与 Solar 映射

| 操作 | 说明 | Solar 现有实现 | 待补充 |
|------|------|----------------|--------|
| **Encoding** | 信息编码入库 | mem_events INSERT | 多模态编码 |
| **Storage** | 持久化存储 | sys_favorites, evo_memory_* | 版本控制 |
| **Retrieval** | 检索调取 | Tantivy + FTS5 | 混合检索 |
| **Consolidation** | 记忆整合巩固 | State Updater (Lite) | 完整巩固机制 |
| **Forgetting** | 选择性遗忘 | 无 | TTL + 重要性衰减 |
| **Transfer** | 跨主体迁移 | context.json 注入 | 多Agent共享 |
